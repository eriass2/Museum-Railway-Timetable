# Vue mount config (from PHP)

Each shortcode emits JSON via `<script class="mrt-vue-config">`.

## `app: "month"`

See `MRT_vue_month_config()` in `inc/public/vue-shortcode-config.php`.

## `app: "overview"`

- `timetableId` (required)

## `app: "wizard"`

See `MRT_vue_wizard_config()` — `stations`, `wizard`, `labels`, `startOfWeek`, etc.
