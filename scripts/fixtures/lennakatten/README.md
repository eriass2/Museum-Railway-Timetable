# Lennakatten fixture tooling

Python and PHP scripts that build and verify `testdata/fixtures/lennakatten/` from PDF sources.

Run from repo root (examples):

```bash
python scripts/fixtures/lennakatten/sync-lennakatten-rail-fixture.py
python scripts/fixtures/lennakatten/generate-lennakatten-extra-timetables.py
python scripts/fixtures/lennakatten/verify-lennakatten-vs-pdf.py
```

See also [testdata/fixtures/lennakatten/README.md](../../../testdata/fixtures/lennakatten/README.md) and [testdata/reference-pdfs/README.md](../../../testdata/reference-pdfs/README.md).
